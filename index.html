Summary: Latitude vs. X

This project will start with this landing page. There are (4) links in the sidebar that have been created from a previous project, which utilized the OpenWeatherAPI and collected data from over 500 cities.

The data reviewed the relationship between the distance to the equator and the weather. The (4) scatterplot graphs review the relationship (Latitude) on 1) Max Temperature, 2) Humidity, 3) Cloudiness, and 4) Wind Speed. The scatterplot graphs were created using Matplotlib.
